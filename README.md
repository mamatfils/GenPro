# GenPro - Script Generator

## 🚀 Déploiement sur Vercel (5 minutes)

### Étape 1 — Crée un compte GitHub
👉 https://github.com → "Sign up" (gratuit)

### Étape 2 — Crée un nouveau dépôt
1. Clique sur "+" > "New repository"
2. Nom : `genpro`
3. Clique "Create repository"

### Étape 3 — Upload les fichiers
1. Clique "uploading an existing file"
2. Glisse-dépose le dossier `genpro` entier
3. Clique "Commit changes"

### Étape 4 — Déploie sur Vercel
1. Va sur 👉 https://vercel.com
2. "Sign up" avec ton compte GitHub
3. Clique "Add New Project"
4. Sélectionne ton dépôt `genpro`
5. Clique "Deploy" → attends 2 minutes ✅

### Étape 5 — Convertis en APK Android
1. Copie l'URL Vercel (ex: `https://genpro-xxx.vercel.app`)
2. Va sur 👉 https://appsgeyser.com
3. Choisis "Website" → colle l'URL
4. Clique "Generate" → télécharge l'APK
5. Installe sur ton Android (active "Sources inconnues" dans Paramètres)

## ⚠️ Important
L'app utilise l'API Anthropic. Pour la production, il faudra
ajouter ta clé API côté serveur (pas directement dans le code client).
